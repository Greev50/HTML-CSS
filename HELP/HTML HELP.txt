title - отобрадающееся имя сайта

meta - тег для характеристик сайта
    *charset - кодировка сайта (UTF-8)
    *name="author" content="**" - Имя автора сайта, ** - имя
    *name="description" content="**" - краткое описание страницы
    *name="keywords" content = "**" - ключевые слова

<link rel="stylesheet" href = "**"> - ссылка на css файл для стилей

h(1-6) - заголовки текста

#Работа с текстом
p - абзац
b - жирный текст, или же strong
i - курсив
strike - перечеркнутый
sub - нижний регистр
sup - верхний регистр
br - перенос на другую строку, одиночный тег
hr - горизонтальная линия
abbr - Аббревиатура, при добавлении атрибута title="**" 
    при наведении на текст высвечивает **
adress - условно для указания адреса (Сиетл, USA)
big - делает текст чуть больше
small - делает текст чуть меньше
pre - дает возможность сохранить все отступы и пробелы текста

#Списки
ul - unnumerated list, ненумерованный список
    *type="**"
        ** = disc, circle, square

ol - ordered list, нумерованный список
    *start="**" - дает возможность установить, 
        с какой цифры будет начинаться подсчет элементов
    *type="**" - изменяет вид нумерации
        ** = 1,A,a,I,i
    
li - list item, элемент списка
img - поставить фотку
    *src="**"
